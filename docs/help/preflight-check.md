# Preflight Check

## Before starting

- 対象Issueと完了条件が明確
- 関連Planの有無を確認
- 影響範囲のdocsを特定

## Before opening PR

- `git diff` を確認
- 検証コマンドと結果を記録
- 回帰リスクとロールバック手順を記載
- 機密情報の混入がないことを確認
